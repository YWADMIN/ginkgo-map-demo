# ginkgo-map-demo

ginkgo-map是Vue的一个高德地图插件，这是使用该插件的一个示例。示例内容如下：

## 显示点、线、面

包括：
- 点Marker
- 折线Polyline
- 多边形Polygon
- 圆Circle
- 矩形Rectangle
- 图片ImageLayer
- 分组显示的控制
- 点、线、面的删除

## 创建点、线、面

包括以下元素的新建和修改：
- 点Marker
- 折线Polyline
- 多边形Polygon
- 圆Circle
- 矩形Rectangle
- 创建后数据的回传

## 回放轨迹

包括：
- 轨迹的不同路段显示不同风格
- 回放过程的控制
